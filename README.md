# logic-gate-circuit-simulations
A collection of basic digital logic gate implementations created using an online digital circuit simulator. Includes circuit designs, truth tables, simulation results, and screenshots for AND, OR, NOT, NAND, NOR, XOR gates.
This repository contains the implementation and simulation of fundamental digital logic gates using an online digital circuit simulation platform. It includes the design and verification of basic combinational logic circuits such as AND, OR, NOT, NAND, NOR, XOR, and XNOR gates. Each project demonstrates the circuit implementation, corresponding truth table, input-output verification, and simulation results through screenshots. This repository serves as a foundational resource for understanding digital electronics, Boolean logic, and logic gate operations, making it suitable for students, beginners, and anyone learning digital circuit design.
